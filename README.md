# DC-DC Boost Converter (Step-Up Converter)

## 📌 Description
This project simulates a DC-DC boost converter using MATLAB/Simulink. The converter increases the input DC voltage to a higher output voltage using controlled switching.

## ⚙️ Features
- Step-up (boost) voltage conversion
- PWM-based switching control
- Duty cycle variation
- Output voltage regulation

## 🛠️ Tools Used
- MATLAB
- Simulink

## ▶️ How to Run
1. Open the .slx file in MATLAB
2. Run the simulation
3. Observe input and output voltages in Scope
